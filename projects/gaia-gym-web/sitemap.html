About

Gaïa Gym is an independent gym offering different solutions and services to physically prepare their subscribers for outdoor and outdoor activities. These solutions and services include facilities such as an outdoor climbing wall, a mountain bike and ski rental system as well as several group lessons with coaches specializing in the physical preparation of customers for outdoor sports such as skiing, trekking, mountaineering and mountain biking.

Objective

The main objective of the project is to create a website for Gaïa Gym that offers activities, equipment, collective classes and workouts. The platform is a web-based app using Javascript and other Web APis so it can be used instantaneously without install and can be accessed through multiple devices such as a desktop, laptop or mobile. This allows for maximum compatibility and accessibility for customers.
